Features

あなたの学習を変える機能

勉強の記録から分析まで、すべてが揃っています。

⏱️
学習タイマー
ストップウォッチ / カウントダウンの2モード対応。バックグラウンドでもタイマーが継続します。
📚
教材管理
教科書・問題集の登録、ページ進捗、問題ごとの正誤記録。ISBNバーコードスキャンで自動メタデータ取得。
📅
カレンダー・ヒートマップ
月間カレンダーにヒートマップ表示。学習量を直感的に把握できます。
📊
詳細レポート
日別 / 週別 / 月別の学習時間内訳、科目別分析、連続学習日数を自動集計。
🎯
目標設定
曜日ごとの学習時間目標を設定。進捗リングで達成率をリアルタイム表示。
📝
テスト管理
テスト日の登録、カウントダウン表示、緊急度バッジで試験対策をサポート。
📋
学習計画
週間計画の作成、実績時間の自動集計、達成率トラッキングで計画的な学習を実現。
☁️
クラウド同期
Firebaseによるクラウド同期。複数デバイスでデータを共有できます。
🔲
ホーム画面ウィジェット
Android 6種 + iOS WidgetKit。ホーム画面で学習状況をひと目で確認。
