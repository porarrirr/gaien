🔒 プライバシーポリシー

個人情報取扱事業者: StudyApp 開発者（以下「開発者」）
お問い合わせ先: toiawasestudyapp@rarari.org

開発者は、個人情報の保護に関する法律（平成15年法律第57号。以下「個人情報保護法」）その他の関連法令を遵守し、ユーザーの個人情報を適切に取り扱います。

1. 取得する情報および利用目的

開発者は、以下の情報を取得し、それぞれ記載の目的の範囲内で利用します（個人情報保護法第17条・第18条）。

  • アカウント情報（メールアドレス・パスワード）: Firebase Authentication を通じたユーザー認証およびクラウド同期機能の提供に利用します。パスワードは Firebase のサーバー上でハッシュ化され、開発者が直接閲覧・取得することはありません。
  • 学習記録データ（科目、教材名、タイマー記録、学習目標、テスト日程等）: アプリ内での学習記録の保存・表示、およびクラウド同期機能をご利用の場合はサーバーへの保管に利用します。
  • 匿名利用統計情報: アプリの品質向上および不具合調査のため、Google Analytics for Firebase を通じて、個人を特定できない形での匿名統計情報（画面遷移、機能利用状況等）を収集することがあります。
  • クラッシュログ: アプリの不具合調査・改善のため、Firebase Crashlytics を通じて、クラッシュ発生時の匿名技術情報（端末機種、OSバージョン、スタックトレース等）を自動収集することがあります。

2. 利用する第三者サービス（SDK）

当アプリは以下の第三者サービスを利用しており、各サービスにおいて情報が処理されます。各サービスの詳細は、それぞれのプライバシーポリシーをご確認ください。

サービス名 提供元 利用目的
Firebase Authentication Google LLC ユーザー認証
Cloud Firestore Google LLC クラウドデータ同期・保管
Google Analytics for Firebase Google LLC 匿名利用統計の収集
Firebase Crashlytics Google LLC クラッシュログの収集・分析

各サービスのプライバシーポリシー: Google プライバシーポリシー

3. 外国にある第三者への提供（個人情報保護法第28条）

クラウド同期機能をご利用の場合、ユーザーの個人データは、以下の条件のもとで外国にある第三者（Google LLC、米国）に提供されます。

  • 提供先の国: アメリカ合衆国（Google LLC の本社所在地。なお、実際のデータ保管場所は Google のインフラにより異なる場合があります）
  • 当該国の個人情報保護制度: 米国には、日本の個人情報保護法に相当する包括的な連邦法は存在しませんが、Google LLC は EU-US Data Privacy Framework の認証を受けており、GDPR に準拠した個人データ保護措置を講じています。
  • 提供先が講じている保護措置: Google LLC は、SOC 2 / SOC 3 認証、ISO 27001 認証を取得しており、データの暗号化（保管時および転送時）、アクセス制御、定期的なセキュリティ監査等の措置を実施しています。詳細は Google Cloud セキュリティ をご参照ください。

4. 第三者提供について

開発者は、以下の場合を除き、取得した個人情報をユーザーの同意なしに第三者に提供・開示することはありません。

  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき
  • 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき
  • 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき

5. 安全管理措置（個人情報保護法第23条）

開発者は、取り扱う個人データの漏えい、滅失または毀損の防止その他の安全管理のため、以下の措置を講じています。

  • 通信の暗号化: アプリとサーバー間の通信は SSL/TLS により暗号化されています。
  • アクセス制御: Firebase Security Rules により、ユーザーは自身のデータにのみアクセスできるよう制御しています。
  • 認証情報の保護: パスワードは Firebase Authentication によりハッシュ化して保管され、開発者が平文のパスワードにアクセスすることはありません。
  • データ保管: クラウドデータは Google LLC が運営するデータセンターに保管され、同社の物理的・技術的セキュリティ措置により保護されています。

6. 開示等の請求手続き（個人情報保護法第33条〜第39条）

ユーザーは、開発者に対して、個人情報保護法に基づき、以下の請求を行うことができます。

  • 保有個人データの利用目的の通知
  • 保有個人データの開示
  • 保有個人データの内容の訂正、追加または削除
  • 保有個人データの利用の停止または消去
  • 保有個人データの第三者への提供の停止

上記の請求を希望される場合は、本人確認のうえ、下記お問い合わせ先までメールにてご連絡ください。合理的な期間内に対応いたします。なお、アプリ内の設定画面から、ご自身のアカウントおよび登録データを直接削除することも可能です。

7. データの保管・削除

クラウド同期機能をご利用の場合、データは Firebase（Google LLC）のサーバーに保管されます。ユーザーはアプリ内の設定画面からいつでもご自身のアカウントおよび登録されたデータを完全に削除することができます。アカウント削除後、サーバー上のデータは速やかに削除されます（ただし、バックアップからの完全消去には一定期間を要する場合があります）。

8. Cookie・端末識別子等の利用

当アプリは Web ブラウザベースのサービスではないため、Cookie は使用しません。ただし、Google Analytics for Firebase および Firebase Crashlytics は、端末を識別するための匿名識別子（Instance ID 等）を利用します。これらの識別子は個人を特定するものではなく、統計分析およびクラッシュ分析の目的にのみ使用されます。
